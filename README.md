# projects

A monorepo containing personal projects — web apps, mobile apps, and more.

## Structure

```
/
├── sourdough-companion/     # Web app — React + Vite + Supabase
└── android-emoji-keyboard/  # Android app — Kotlin
```

## Projects

### 🍞 sourdough-companion
A web app for sourdough bread bakers.
**Stack:** React, TypeScript, Vite, Supabase
**Path:** [`sourdough-companion/`](./sourdough-companion)

---

### ⌨️ android-emoji-keyboard
A custom emoji keyboard for Android.
**Stack:** Kotlin, Android SDK
**Path:** [`android-emoji-keyboard/`](./android-emoji-keyboard)

---

## Adding a New Project

Each new project gets its own top-level folder:

```bash
/your-new-project/
```

Then add it to the table above.
